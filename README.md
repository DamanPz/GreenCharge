# GreenCharge

**A Smart EV Charging Management System aligned with SDG 7: Affordable & Clean Energy.**

GreenCharge is a sustainable web-based initiative designed to improve electric vehicle (EV) charging infrastructure and visibility in Bengaluru. This project aims to make EV adoption easier by providing real-time data on station availability and renewable energy usage.

## Features (Phase 1)
Currently, the project runs as a **local prototype** with the following features:

- **Interactive Map**: View charging station locations across the city.
- **Admin Dashboard**: Manage stations, view system capacity, and monitor status.
- **Analytics**: Visual charts for energy usage and station status using Chart.js.
- **Search & Filter**: Find stations by name, location, or availability.
- **Responsive Design**: Professional UI/UX with a clean green sustainability theme.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js (Mock API for Phase 1)
- **Mapping**: Leaflet + OpenStreetMap
- **Visualizations**: Chart.js, DotLottie
